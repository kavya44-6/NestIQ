# NestIQ

NestIQ is a trust-centric real estate platform developed using React, Spring Boot, and MySQL.

## Features

- Role-based authentication (Admin, Owner, Agent, Customer)
- Property listing and management
- Trust Score system
- Rental price prediction
- AI-powered property recommendations
- Visit scheduling
- Property verification workflow
- JWT-based security

## Tech Stack

### Frontend
- React
- Vite
- CSS

### Backend
- Spring Boot
- Java

### Database
- MySQL
